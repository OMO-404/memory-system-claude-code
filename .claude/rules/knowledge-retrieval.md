# 知识检索流程

## 指令识别

用户提问技术问题时触发（如"OOM怎么解决"、"关联sku的sql"）

## 处理步骤

### 1. 匹配主题

根据问题关键词匹配对应主题目录：

| 关键词  | 主题目录 |
|------|------|
| {主题} | {路径} |


### 2. 在主题目录下搜索

**必须同时使用**：
- **Glob** - 搜索文件名（如 `*OOM*`、`*sku*`）
- **Grep** - 搜索文件内容

### 3. 读取内容

使用 Read 工具读取找到的文件

### 4. 输出结果

- 找到 → 输出内容
- 未找到 → 提示"知识库中未找到相关内容"
