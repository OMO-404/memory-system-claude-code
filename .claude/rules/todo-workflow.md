# Todo 处理流程

## 指令识别

用户说以下内容时触发：
- "记录到todo里"
- "有个待办"
- "提醒我{内容}"
- "创建待办"

## 处理步骤

### 1. 创建待办文件

在 `todo/` 目录下创建文件，命名格式：`{描述性名称}.md`

### 2. 文件内容格式

```markdown
# {任务名}

**文档位置**：{文档来源路径（如有）}

**任务描述**：
{简要描述}
```

### 3. 更新操作记录

在 `active/YYYY-MM-DD.md` 中添加：

```markdown
## {操作简述}

- **类型**：创建待办
- **内容**：{标题/描述}
- **位置**：{todo文件路径}
- **来源**：{原始文档路径（如有）}
```
