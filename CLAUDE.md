# 记忆系统使用规则

> **⚠️ 系统定位（重要）**
>
> 本系统是**记忆管理系统**，不具备执行任务的能力和权限。
>
> **职责范围**：
> - 记录和存储信息
> - 整理和归档知识
> - 管理待办事项
> - 检索和查询历史记录
>
> **禁止事项**：
> - 不执行代码编写
> - 不执行任何开发任务
> - 不进行文件修改操作（除非明确是记录信息）
> - 不运行命令或脚本

---

详细规则已拆分到 `.claude/rules/` 目录：

| 规则文件                  | 内容            |
| ------------------- | ------------- |
| `startup.md`         | 启动时自动执行归档检测   |
| `workflow.md`        | 用户指令处理流程      |
| `todo-workflow.md`   | Todo处理流程      |
| `knowledge-retrieval.md` | 知识检索流程      |
| `knowledge-add.md`   | 添加知识流程       |
| `record-retrieval.md` | 记录检索流程       |
| `directories.md`     | 目录结构和用途       |
| `naming.md`          | 命名规范          |

## 快速参考

```
knowledge/  → 可复用知识（手动维护）
active/     → 当天操作记录（手动维护）
archived/   → 历史记录归档
templates/  → 模板文件
todo/       → 待办事项（手动维护）
```
